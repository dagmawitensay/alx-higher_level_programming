How to use 0-add_integer.py
===========================

This library only has one function called ``add_integer()``.

Numbers
=======
``add_integer()`` returns the sum of its arguments.
For numbers it's equivalent to using the + operator.

::
    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(2, 4)
    6
    >>> add_integer(4.5, 28.5)
    32

Non-Numbers
===========
``add_integer()`` raises a TypeError for arguments that are not integers or floats.

::
    >>> add_integer("hello", 5)
    Traceback (most recent call last):
        ...
    TypeError: a must be an integer

Checking for string argument passed to b
===========
    >>> add_integer(5, "hello")
    Traceback (most recent call last):
        ...
    TypeError:  b must be an integer


Checking for only one argument
===========
``add_integer()`` adds 98 to the argument if only one argument is provided.

::
    >>> add_integer(5)
    103

Checking for negative numbers:
==============================
``add_integer()`` should correctly add negative numbers as well.

::
    >>> add_integer(-4, -5)
    -9
